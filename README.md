# SmileCraft Dental Clinic — Website

A professional, fully responsive dental clinic website for **Dr. Pooja Vijapure, BDS** — SmileCraft Dental Clinic, Pune.

## 🚀 GitHub Pages Deployment

### Steps to Deploy:

1. **Create a new GitHub repository**
   - Go to [github.com](https://github.com) → New Repository
   - Name it: `smilecraft-dental` (or any name you prefer)
   - Make it **Public**
   - Click **Create Repository**

2. **Upload the file**
   - Click **Upload Files**
   - Drag and drop `index.html`
   - Click **Commit Changes**

3. **Enable GitHub Pages**
   - Go to **Settings** → **Pages**
   - Under **Source**, select `main` branch → `/ (root)`
   - Click **Save**

4. **Your site is live!**
   - It will be available at: `https://yourusername.github.io/smilecraft-dental`
